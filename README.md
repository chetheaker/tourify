# Tourify

Road trip planner app that allows users to schedule and visualise trips by adding destinations to a map as well as creating day-by-day itineraries.

## Tech Stack

| Front End           |  Back End  |
| ------------------- | :--------: |
| React               |  Express   |
| Google Maps API     |  MongoDB   |
| Google Calendar API | PassportJS |
